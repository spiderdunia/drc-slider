DRC Slider for practical Interview Test.


== Installation ==

1. Download From https://github.com/spiderdunia/drc-slider
2. Unpack file package using your favorite zip software.
3. extract .zip file to your wordpress plugin directory on your server.
4. Login to wordpress admin and go to plugin tab and activate "DRC Slider" Plugin.

== Upgrade ==

Upgrade "DRC Slider" is easy, just replace all plugin file in plugin directory with new version. No need to re-configure any other options in "DRC Slider" tab.
or You can goto update plugin list and update your plugin.
